# Gaming-Tagebuch
Die App ist für da, dass man bereits gespielte Spiele eingetragen kann und sozusagen ein Tagebuch führen kann.

## Features
Die Struktur eines Eintrags ist folgendes:
  - Name des Spiels
  - eine Bewertung vom Benutzer (mit Schulnoten)
  - kurzes Fazit (nicht pflicht)
  - Die Plattform (PC,XBox,Playstation,Nintendo) auf dem das Spiel gezockt wurde
  - Das Datum an dem das Spiel begonnen wurde und ein Datum an dem das Spiel beendet wurde

- Einträge sollten:
  - gespeichert werden
  - nach den abspeicher bearbeitetbar, löschbar und sortierbar sein

- Wenn es mind. ein Spiel gibt, kann man eine Statistik durchlesen mit den wichtigsten Daten 
  z.B Was ist die meist verwendete Plattform, beste und schlechteste Bewertung...

- Eine Suchfunktion gibt es auch und Klein-/Großschreibung wird ignoriert


## Verwendete Technologien
